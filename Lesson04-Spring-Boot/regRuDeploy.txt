1. ```ssh root@123.123.123.11```
2. yes
3. copy, paste password (когда вводите пароль в линукс системах - символы не отображаются)
4. ```apt update``` - обновить пакеты, которые находятся на образе
5. ```apt install openjdk-11-jdk```
6. java -version (check)
7. Кроме JVM ничего не нужно. Теперь нужно jar файл скопировать на сервер (виртуалку) на винде программа для этого winscp 
8. создадим директорию на сервере mkdir /opt/webapp
9. Идем в директорию с приложением, набираем команду scp  
10. ```scp lesson4-spring--boot-0.0.1-SNAPSHOT.jar root@123.123.123.123:/opt/webapp/app.jar```
11. Возвращаемся в консоль - запускаем ```java -jar /opt/webapp/app.jar```
12. В браузере 123.123.123.123:8080/mvc-app/user (должно открыться приложение)
13. Приложение будет работать, пока работает консоль. Чтобы оно не было жестко привязано к консоли - нужно научить наше приложение запускаться в виде
фонового сервиса (демона)
14. В консоли - нужно создать пользователя. sudo useradd spring_boot 
15. Дать доступ этому пользователю на файлы нашего приложения ```sudo chown spring_boot:spring_boot /opt/webapp/app.jar```
16. Дать доступ на запуск приложения ```sudo chmod 500 /opt/webapp/app.jar```
17. Создать дескриптор приложения ```sudo nano /etc/systemd/system/spring-boot-app.service``` (создаем новый файл и открываем его в nano)
